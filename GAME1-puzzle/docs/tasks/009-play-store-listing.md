# Task 009: Google Play ストア提出物を整備（テキスト・素材ドラフト）

最終更新日: 2026-04-18
対応タスク: #10 (Google Play ストア提出物を整備)
仕様書順: `docs/SPECIFICATION.md §15 項目 7`
コスト: Play Console 登録 (US$25 一回払い) は **別タスク**。本タスクは無料範囲のドラフト整備まで

## 1. 目的・スコープ

Google Play にアプリを申請するために必要な各種テキスト・素材のうち、無料で整備できる部分を 1 ファイルに集約する。Play Console 登録と実機スクリーンショット撮影以外の要素をすべてドラフト化し、オーナーが Play Console 上にコピペするだけの状態にする。

### 対象（本タスク）

- ストア掲載テキスト一式のドラフト
  - アプリ名（最大 30 文字）
  - 簡単な説明（最大 80 文字）
  - 詳細な説明（最大 4000 文字）
- コンテンツレーティングアンケートの想定回答
- ターゲット層・カテゴリの推奨値
- 連絡先ポリシー（メールアドレス、サポート URL）
- プライバシーポリシー URL（Task #9 で用意済みの `docs/PRIVACY_POLICY.md` をブログへ掲載する前提）
- Feature graphic（1024×500）の制作方針
- スクリーンショット取得の手順書

### 対象外（別タスク／オーナー対応）

- Play Console アカウント登録（US$25、本人確認）
- 実機／エミュレーターでのスクリーンショット撮影（Task #13 に含める）
- Feature graphic 画像そのものの作成（Canva 等で制作。雛形テキストは本タスクで用意する）
- 本番 AdMob ID（Task #6）
- 本番課金（Task #8）

## 2. 前提と制約

- アプリ名称は「GAME1 Puzzle」で確定（`AndroidManifest.xml` / `strings.xml` / README と一致）
- Android 版のみ。iOS は対象外
- 日本語圏が主ターゲット、英語は二次
- プライバシーポリシーはオーナーのブログに掲載する方針（Task #9 確定）
- 広告: AdMob（テスト ID → Task #6 で本番差し替え）
- 課金: 未実装（Task #8）。初版提出時は「広告のみ・課金なし」で申請する可能性あり

## 4. コンテンツ方針

### 4.1 アプリ名

「GAME1 Puzzle」。30 文字制限に対して 12 文字で十分。日本語名は不要（シンプルさ優先）。

### 4.2 簡単な説明（最大 80 文字）

**ドラフト**: 「懐かしい 15 パズル。1〜15 の数字タイルを順に並べて、最小手数でクリアしよう。」（38 文字）

**狙い**:
- 「15 パズル」キーワードで検索流入
- 「最小手数」でコンテンツの遊び（ベストスコア保存）を示唆
- 過度な訴求語（"無料" "今すぐ"）を使わず Google Play ポリシーの誤解訴求リスクを下げる

### 4.3 詳細な説明（最大 4000 文字）

段落構成（ドラフトは `docs/STORE_LISTING.md` に保存）:

1. ゲーム概要（2〜3 文）
2. 遊び方（タップ + スワイプ）
3. 特徴（ベストスコア保存、自動再開、オフライン動作、シンプル UI、日本語対応）
4. 広告表示について（バナー＋クリア時のインタースティシャル、広告削除オプション予定）
5. データとプライバシー（localStorage のみ、ポリシーへのリンク）
6. 動作環境（Android 7.0+、オンライン／オフライン）
7. 今後の予定（ヒント機能、履歴、テーマ）
8. フィードバック窓口（メール）

### 4.4 カテゴリ

**推奨**: `ゲーム › パズル`。
Play Console の category: `GAME_PUZZLE`。

### 4.5 ターゲット年齢・コンテンツレーティング

- ターゲット年齢: 全年齢（3+）
- 暴力・性的表現・薬物・賭博要素: 無
- ユーザー間通信: 無
- 位置情報: 無
- 広告: あり（AdMob バナー + インタースティシャル、すべて家族向け）
- 課金: 広告削除のみ（Task #8 実装後）

IARC 設問への想定回答は `docs/STORE_LISTING.md` で網羅する。

### 4.6 連絡先

- メール: `<!-- OWNER_INPUT_NEEDED: 公開メールアドレス -->`（auto-memory では `kuresukensuruna@gmail.com` が記録済みだが、ストア公開用の別アドレス推奨）
- ウェブサイト: オーナーのブログ URL（プライバシーポリシー掲載先と同じ）

### 4.7 素材方針

- **アイコン**: Task #5 で確定した `assets/icon.png`（512×512）をそのまま使用
- **Feature graphic** (1024×500): ブランド配色（ダーク系、タイル配列ビジュアル + 「GAME1 Puzzle」文字）で Canva 等で作成。本タスクでは Canva テンプレートに流し込むコピー案だけ提示
- **スクリーンショット**: Task #13 通しプレイ時にエミュレーターで取得する（既に `docs/HARNESS_RUNBOOK.md` §46 に取得手順あり）。最低 2 枚、推奨 4 枚
